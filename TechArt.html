Landscape Grass System

Grass Color

Path Height

Path Normal

Built inside of Unreal Engine 5 (all assets made by me)

All 3 textures are sampled with top-down projection

Landscape uses UEs landscape material layers to height blend the path textures over the grass. Additionally the blending takes into account physics materials for linkage with other systems.

Grass and foliage is automatically instanced

The material that the foliage uses gradually adjusts hue and saturation according to a vertical gradient. All foliage also uses vertical split normals to mimic the same shading as the ground.

The result is that the foliage blends into the ground below it. Your eyes cannot follow a single blade of grass to its root. Very fluffy and stylized.
Level designers and environment artists can easily paint where the path should go. The system was packaged into a zip folder and given to some collegues. On the right is a game jam game Lucas Foxworthy made with my package.

Ice Shader

Ice Normal

Perlin Noise

What started as an environment for a passion project pivoted towards refining an interesting shader setup.

This ice shader mainly uses parallax mapping with a noise texture The noise is used for colorization

What enables the true ice effect is that the parallax mapping uses non uniform depth, the depth is scaled by the noise texture.

As a final layer of ice-ness, an icy normal map is used alongside some direct specular
