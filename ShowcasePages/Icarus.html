Icarus Ignition

General Info

Rocket jumping prototype game made in Unity.

Utilizes AwesomeX's Fragsurf Source Physics for air strafing and sliding.

Game is intended to be played fast, to get the best times on the levels. A ranking system is displayed at the end of the level, and it is configured for the level using a scriptable object

Data-Driven Level Select

The game needed to be able to support a dynamic collection of levels.

Designers can register their levels with the level select menu, giving them names, thumbnails, and difficulties

Then the level select menu generates a clickable level select button for each registered level.

Score Ranking System

Part of the level creation toolkit I made for myself

The level manager is a singleton that represents the gamemode's rules

It also contains data entry fields for setting and tweaking score rankings at the end of the level

Time Rewind

The player can rewind time

Uses a custom circular array buffer to store time data (an interface)

Time data objects internally manages storing and applying data.

The circular array buffer manages the entire collection of time data, it stores lists of time data for each frame, and applies them when moving backward through time.

Objects that need to track their variables over time, register themselves with the system using the factory pattern.

Toon Explosion

A rocket jumping game is incomplete without explosions.

This is a low-scope explosion using the Unity VFX Graph.

Using a cloudlike mesh made with meta-balls.

All that happens aside fromthe spherical velocity is a color shift and a voronoi dissolve.
