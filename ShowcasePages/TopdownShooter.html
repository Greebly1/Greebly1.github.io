Fool's Gold (WIP)

This is an early WIP topdown/third person game I am making in Unity

I've had a lot of time to polish the gameplay systems

First of all, all player input is seperated into state machines and decoupled completely, then a designer created event system specifies how they can communicate. For instance, the UI select event can tell the locomotion controller to attempt to possess a gameobject it selects

Second, the camera controller can switch between dynamic top down, and third person at runtime

All pawns use root motion to move and implement the interact interface

Lastly, the projectiles use basic object pooling
