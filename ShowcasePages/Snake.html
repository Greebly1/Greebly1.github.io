Snakelike Dungeon Generator

Built inside of Unity Engine for rougelike tank game concept

Essentially, play a random game of snake, then after the snake reaches a certain size, turn it's body into rooms for a rougelike dungeon.

Algorithm

Start at (0,0), and place a room

1. Generate a random direction

2. Open the door in that direction

3. Move in that direction

4. Place a room

5. Open the door from the direction you just moved

6. Repeat

The only complicated part of this project was the 2D list, I wanted a grid that would expand as the snake moved beyond its indices

Tank Drifting Locomotion

Built inside of Unity Engine for roguelike tank game same as above

Does not use rigid bodies, all physics are handled within the locomotion system for complete control over the way these move

Acceleration is used as the middle man between the controller and the velocity they want to move at

Ai's predict where they are headed and adjust the throttle acceleration and turning according to their goal position

Fool's Gold (WIP)

This is an early WIP topdown/third person game I am making in Unity

I've had a lot of time to polish the gameplay systems

First of all, all player input is seperated into state machines and decoupled completely, then a designer created event system specifies how they can communicate. For instance, the UI select event can tell the locomotion controller to attempt to possess a gameobject it selects

Second, the camera controller can switch between dynamic top down, and third person at runtime

All pawns use root motion to move and implement the interact interface

Lastly, the projectiles use basic object pooling
