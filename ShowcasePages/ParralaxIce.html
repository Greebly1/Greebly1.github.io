Ice Shader

Ice Normal

Perlin Noise

What started as an environment for a passion project pivoted towards refining an interesting shader setup I learned from Ben Cloward.

This ice shader mainly uses parallax mapping with a noise texture

What enables the true ice effect is that the parallax mapping uses non uniform depth, the depth is scaled by the noise texture.

As a final layer of ice-ness, an icy normal map is used alongside some wet roughness values
