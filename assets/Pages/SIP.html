Modular Procedural Generation [MPG]

Modular Procedural Generation is a specification for improving modern proc-gen systems to support extensibility and modularity. Currently this technique is only theoretically possible with plugins for Houdini-Unreal.

MPG enables a workflow I call 'point-to-point' level creatuin, which is an innovative new way to approach 3D environment authoring.
